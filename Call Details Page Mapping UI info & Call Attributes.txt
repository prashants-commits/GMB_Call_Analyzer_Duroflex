## Important Implementation Notes

- Convert all scores:
  - `1 → LOW`
  - `2 → MEDIUM`
  - `3 → HIGH`
  - `4/5 → HIGH` (for purchase readiness)

- Derived fields:
  - Intent Labels → from score buckets
  - Experience → from NPS
  - Funnel highlight → from `4_Funnel_Analysis_Stage`

- Location format:Locality + City + State


- Duration formatting: seconds → mm:ss
